  Size:

      Not empty.
  Number of String in the file:
      None.			            [single] [if !emptyfile] [property noOccurences]
      One.				                [if !emptyfile]
      Many.				                [if !emptyfile]
  Number of String in one line:
      One.				                [if !noOccurences && !emptyfile]
      Many.			            [if !noOccurences && !emptyfile]
  Position of the String in the file:
      First line.		        [single] [if !emptyfile]
      Last line.		        [single] [if !emptyfile]

  Presence of enclosing quotes:
      Not enclosed.			        [if !zerovalue]
      Enclosed.
      Incorrect.		      [error]
  Presence of blanks:
      None.
      One.				              [if !zerovalue]

  Presence of quotes within the String:
      None.
      One.				              [if !zerovalue]
      Many.			              [if !zerovalue && Stringlengthgt1]
